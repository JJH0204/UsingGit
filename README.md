- - -
> Homebrew 설치
- [Homebrew 설치링크](https://brew.sh) 에서 알려주는 설명대로 터미널을 통해서 설치하는게 일반적이지만 Mac OS의 경우 [Homebrew 릴리즈 다운로더](https://github.com/Homebrew/brew/releases/latest) 로 설치하는게 마음편하다.
- 다운로더 실행 후 설정 변경없이 다음 누르기만하면 되서 편하다.
